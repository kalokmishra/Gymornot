## Contributor Covenant Code of Conduct

Our community is dedicated to providing a welcoming and inclusive experience for everyone. All contributors are expected to follow the Code of Conduct.

Be kind and respectful. Harassment, discrimination, and hateful language are not acceptable. Treat people with patience and assume good intent.

If you experience or witness unacceptable behavior, please contact the maintainers so we can address the issue promptly. You can reach out via a private GitHub issue (mark as "private") or email the maintainer listed in `.github/CODEOWNERS`.

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/).

Reporting
- To report issues privately, create a new issue and set the visibility to private (if your GitHub plan supports it), or email the maintainer.
- Maintain a respectful and factual tone in reports; include timestamps, links, and any relevant context.

Enforcement
- Project maintainers and CODEOWNERS may moderate or close issues, and will follow the Contributor Covenant procedures when investigating reports.
